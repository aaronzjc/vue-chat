# 说明

> 使用vue.js, SUI, WebSocket开发的的WebAPP.

## 构建

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run all tests
npm test
```
